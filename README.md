# 📊 IBM Data Analytics Capstone Project
# 👩‍💻 Author - Jesna Salim
End-to-end Data Analytics project using Python, SQL, Excel, Plotly &amp; Dash

📌 Project Overview

This is an end-to-end Data Analytics project completed as part of the IBM Data Analytics Certification.

The project focuses on:

- Data Collection using APIs

- Data Cleaning & Preparation

- Exploratory Data Analysis (EDA)

- Feature Engineering

- Interactive Dashboard Development

🛠️ Tools & Technologies

- Python (Pandas, NumPy)

- SQL

- Excel

- Plotly

- Dash

📂 Project Components
🔹 Data Collection

Extracted job market data using APIs

Automated job count extraction by location & technology

Stored structured datasets for analysis

🔹 Data Preparation

Cleaned dataset with 65k+ records

Handled missing values

Encoded categorical variables

🔹 Feature Engineering

Created normalized compensation metrics

Grouped categories for better insights

🔹 Exploratory Data Analysis

Salary variation by role & geography

Impact of experience & job satisfaction

Outlier detection in salary distribution

🔹 Dashboard & Visualization

Built using Plotly & Dash

Compared trends across time periods

Visualized:

- Job trends

- Technology usage

- Salary insights

📈 Key Insights

- 💰 High Salary Skills: Python & Swift offer highest compensation

- 📊 High Job Volume: C & Java dominate job postings

- 🗄️ Database Importance: SQL remains a critical skill

🚀 Future Trends: Growth in AI, Rust, and modern databases
